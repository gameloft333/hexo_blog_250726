## 📋 3步搞定，小白也能轻松上手

### 第1步：复制项目到你的账户 📂

首先，你需要一个GitHub账号（免费注册）。

1. 打开这个链接：https://github.com/senshinya/MoonTV
2. 点击右上角的 `Fork` 按钮
3. 项目就复制到你的账户了

**就这么简单！** 连代码都不用写一行。

![Fork项目](https://images.unsplash.com/photo-1556075798-4825dfaaf498?ixlib=rb-4.0.3&auto=format&fit=crop&w=800&q=80)

### 第2步：免费部署到云端 ☁️

接下来用Cloudflare Pages免费托管（永久免费，不用担心费用）：

1. 登录 Cloudflare，进入 Pages 页面
2. 选择 `连接到Git`，授权你的GitHub
3. 选择刚才复制的 MoonTV 项目
4. 填写构建设置：
   - 构建命令：`pnpm install --frozen-lockfile && pnpm run pages:build`
   - 输出目录：留空即可

点击部署，等待几分钟就完成了！

### 第3步：个性化设置 ⚙️

为了更好的体验，建议添加这些环境变量：

| 设置项 | 推荐值 | 作用 |
|--------|--------|------|
| SITE_NAME | 我的影视库 | 自定义网站名称 |
| PASSWORD | 123456 | 设置访问密码（可选） |
| ANNOUNCEMENT | 仅供学习交流使用 | 添加公告说明 |

**重点来了！** 设置完成后，你就拥有了一个专属的影视平台！
