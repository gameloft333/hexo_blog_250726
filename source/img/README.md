# Gens Studio 图片资源

这个目录用于存放网站的图片资源，包括：

- favicon.png - 网站图标
- apple-touch-icon.png - Apple设备图标
- logo.png - 网站Logo
- avatar.png - 作者头像

## 建议的图片规格

- favicon.png: 32x32px 或 16x16px
- apple-touch-icon.png: 180x180px
- logo.png: 建议200x60px左右
- avatar.png: 建议200x200px

## 注意事项

请确保图片文件大小适中，以提高网站加载速度。
